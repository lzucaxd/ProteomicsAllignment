# Final report assets (index)

This folder is an **index** for paper- and slide-facing outputs. Large binaries are **not duplicated** here by default; paths below point to canonical locations under `reports/` and `presentation_materials/`.

## Primary numeric tables

| Asset | Canonical path |
|-------|----------------|
| Method × task summary | `../benchmark_master/benchmark_results/comparison_summary.csv` |
| Disconnect / “geometry vs FC” style metrics | `../benchmark_master/benchmark_results/disconnect_scores.csv` |
| Tiered / annotated summary (if present) | `../benchmark_master/benchmark_results/comparison_summary_tiered.csv` |
| LLM / paper bundle (Markdown) | `../benchmark_master/final_tables/BENCHMARK_NUMBERS_MASTER_FOR_LLM.md` |
| LaTeX snippets (if present) | `../benchmark_master/final_tables/*.tex` |
| Consistency / audit notes | `../benchmark_master/final_tables/paper_consistency_audit.md` |

**Regenerate** the master Markdown bundle:

```bash
Rscript --vanilla scripts/benchmark/build_final_benchmark_tables.R
```

(from repo root; see `scripts/benchmark/README.md` if the script path differs on your branch).

## Figures

See **`FIGURE_MANIFEST.md`** in this folder for report-facing figures vs exploratory outputs.

**Report-style figures** (regenerated by R):

```bash
Rscript --vanilla scripts/presentation/generate_report_figures.R
```

Outputs typically under `presentation_materials/figures/report/` (**gitignored** — regenerate after clone).
