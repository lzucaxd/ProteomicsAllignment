# Curated annotations (tracked in git)

Small tables that define **subtypes**, **tissue**, **disease**, or **sample ↔ label** mappings for benchmark tasks. Large matrices and PSM outputs stay under **`data/results/`** (gitignored).

| Path | Role |
|------|------|
| **`cptac/PDC000120/gene_matrix_subtype_mapping.csv`** | CPTAC breast (PDC000120): matrix column IDs ↔ PAM50 / sample-type fields used by **`configs/tasks/breast_subtype.yaml`**. Regenerated by **`data/scripts/build_PDC000120_subtype_mapping.py`** (also mirrors a copy next to `gene_matrix.csv` under `data/results/PDC000120/` when that folder exists). |
| **`../biospecimen/`** | CPTAC clinical / biospecimen exports and label-mapping spreadsheets used by **`build_PDC000120_subtype_mapping.py`**. |
| **`../ccle/ccle_breast_subtype_annotations_v2.csv`** | Curated CCLE breast Luminal / Basal / HER2 line list for subtype and BvL tasks. |
| **`../processed/ccle_breast_subtype_annotation_processed.csv`** | Derived from the CCLE CSV via **`scripts/benchmark/process_ccle_annotations_v2.R`** (HER2 handling, CAL120, etc.). |

**Adding new studies:** mirror this layout, e.g. `data/annotations/cptac/<PDC_ID>/…`, point **`subtype_mapping`** (or equivalent) in **`configs/preprocessing/*.yaml`** and task YAMLs at the new file, and extend preprocessing metadata code if needed.
