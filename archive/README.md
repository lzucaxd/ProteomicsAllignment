# Archive

This directory holds **legacy or duplicate outputs** moved during repository cleanup so nothing is deleted outright.

| Path | Note |
|------|------|
| `report_figures_legacy/` | Older slide/report PNG+PDF set; canonical report figures live under `presentation_materials/figures/report/` (see `scripts/presentation/generate_report_figures.R`). |

Add dated subfolders here when retiring large or confusing duplicates.

**Notebooks:** exploratory notebooks live under **`notebooks/exploratory/`** (see `notebooks/exploratory/README.md`). Do not archive vendor notebooks under `models/celligner-master/`.
